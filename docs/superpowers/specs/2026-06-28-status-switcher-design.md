# 任务编辑窗口 - 状态切换功能设计

**日期**：2026-06-28  
**状态**：已批准

## 概述

在任务编辑模态框中添加状态分段切换控件，允许用户点击按钮将任务在"待办"、"进行中"、"已完成"三个状态间快速切换，无需拖拽到目标列。

## UI 布局

在编辑窗口顶部（标题输入框上方）添加三段式水平分段控件：

```
┌─────────────────────────────┐
│  待办  │  进行中  │  已完成  │  ← 分段控件（新增）
├─────────────────────────────┤
│  任务标题...                 │  ← 标题输入框
├─────────────────────────────┤
│  任务描述...                 │  ← 描述文本域
├─────────────────────────────┤
│  [四象限选择器]（仅待办显示） │
├─────────────────────────────┤
│  [取消]  [删除]  [保存]      │  ← 操作按钮
└─────────────────────────────┘
```

- 三个按钮等宽排列
- 当前任务所在状态按钮高亮（蓝色背景/主题色）
- 其余按钮显示为灰色/非激活态
- 与看板列颜色风格一致
- 适配深色/浅色主题

## 交互行为

1. **点击切换**：点击目标状态按钮，立即调用 `Store.move()` 更新任务 `column` 字段
2. **窗口保持打开**：切换后编辑窗口保持打开，用户可继续编辑标题、描述等其他字段
3. **即时保存**：状态切换立即生效并重新渲染看板列，无需额外点击"保存"按钮
4. **completedAt 副作用**：切换到"已完成"列时自动标记 `completedAt` 时间戳；从"已完成"列切出时自动清除 `completedAt`。复用 `Store.move()` 的已有逻辑
5. **四象限处理**：切换到非"待办"状态时隐藏象限选择器 UI，但保留任务对象的 `quadrant` 字段值不动；切回"待办"时恢复显示
6. **番茄钟锁定**：如果任务正被番茄钟锁定，状态切换控件应禁用/不可点击（与拖拽限制一致）

## 实现要点

### 文件修改

所有修改集中在 `/index.html`：

### HTML 结构（第 1139-1181 行区域）

在编辑模态框内，标题输入框上方添加分段切换控件：

- 一个容器 `div` 包含三个按钮
- 每个按钮绑定点击事件
- 当前状态按钮添加 `.active` class

### CSS 样式

- 分段控件使用 flexbox 布局，三个按钮等宽
- 激活状态使用 CSS 变量 `--primary` / `--primary-light`
- 非激活状态使用 `--bg-secondary` / `--text-secondary`
- 过渡动画：`transition: background-color 0.2s, color 0.2s`
- 被番茄钟锁定时添加 `opacity: 0.5; pointer-events: none;`

### JavaScript 逻辑（Modal 模块，第 2348-2507 行区域）

新增函数 `Modal.switchStatus(newColumn)`：

1. 获取当前编辑任务 ID（`_editingTaskId`）
2. 调用 `Store.move(taskId, newColumn)` — 复用已有状态迁移逻辑（含 `completedAt` 处理）
3. 更新分段控件 UI 高亮状态
4. 根据新状态决定是否显示/隐藏四象限选择器
5. 重新渲染受影响的看板列

修改 `openEdit()` 函数：
- 打开编辑窗口时初始化分段控件状态
- 如果任务被番茄钟锁定，禁用分段控件

修改 `openCreate()` 函数：
- 新建任务时，分段控件显示当前目标列，所有按钮可点击
- 点击切换仅更新隐藏的 `taskFormColumn` 字段值，不涉及 Store 操作（任务尚未创建）
- 点击"保存"时，任务将以最终选定的状态列创建
- 四象限选择器根据当前选中的状态（而非切换前状态）决定显隐

### 与已有逻辑的复用

- `Store.move(taskId, targetColumn)` 已处理：
  - 切换到 `completed` → 自动设置 `completedAt`
  - 从 `completed` 移出 → 自动清除 `completedAt`
  - 交叉拖拽时的 reorder
- `Timer.isTaskLocked(taskId)` 已用于检测番茄钟锁定

6. 番茄钟锁定的任务打开编辑窗口时，状态切换控件应禁用
7. 深色/浅色主题下样式正常
8. 移动端触摸交互正常
9. 新建任务时可点击状态切换按钮修改目标列，保存后任务创建在最终选定的列中
